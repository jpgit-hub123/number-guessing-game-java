# number-guessing-game-java 

# Console - based number guessing game built using java.
A Java console game where the user tries to guess a randomly generated number.

# How to Run
1. Compile
   javac NumberGuessingGame.java
2. Run
   java NumberGuessingGame

# Features 
- Random Number generation
- User input handling
